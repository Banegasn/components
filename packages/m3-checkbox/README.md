# M3 Checkbox

Material Design 3 expressive checkbox component.

## Installation

```bash
npm install @banegasn/m3-checkbox
```

## Usage

```html
<script type="module" src="@banegasn/m3-checkbox"></script>

<m3-checkbox></m3-checkbox>
<m3-checkbox checked></m3-checkbox>
<m3-checkbox disabled></m3-checkbox>
```

## API

### Properties
- `checked`: Boolean. Whether it is checked.
- `indeterminate`: Boolean. Whether it is indeterminate.
- `disabled`: Boolean. Whether it is disabled.

### Events
- `checkbox-change`: Fired when the state changes.
