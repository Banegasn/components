# @banegasn/m3-fab-menu

Material Design 3 FAB Menu web component. Replaces speed dial with a more expressive menu interaction.

## Installation

```bash
npm install @banegasn/m3-fab-menu
```

### Via CDN (No build step)

```html
<script type="module" src="https://cdn.jsdelivr.net/npm/@banegasn/m3-fab-menu/+esm"></script>
```

## Usage

```html
<m3-fab-menu>
  <!-- Menu Items go here -->
</m3-fab-menu>
```
