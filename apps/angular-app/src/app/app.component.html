Svelte Components

Svelte components can be integrated using a wrapper or build configuration. This demo shows the monorepo structure - for full Svelte integration in Angular, you would typically use a custom build setup or web component wrapper.

Note: The Svelte component package is available in the workspace and can be used in Svelte-based apps or wrapped as web components for use in Angular.

Monorepo Structure

This project demonstrates:

  • Turborepo for build orchestration
  • PNPM workspaces for dependency management
  • Multi-framework component libraries (Lit, Svelte)
  • Angular application consuming workspace packages
  • TypeScript configuration inheritance
  • Shared build and dev scripts
