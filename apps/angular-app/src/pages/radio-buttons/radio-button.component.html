Radio Buttons

Radio buttons allow users to select a single option from a group of options. They are best used when users need to see all available options.

Basic Usage

Radio buttons allow users to select one option from a group. They follow Material Design 3 specifications with smooth animations and accessibility support.

Default Radio Buttons

Radio buttons start in the unchecked state by default.

<m3-radio-button name="group" value="1"></m3-radio-button>
<m3-radio-button name="group" value="2" checked></m3-radio-button>

Radio Groups

Radio buttons with the same name attribute form a group where only one can be selected at a time.

Theme Selection

Example of a radio group for theme selection:

Selected: {{ selectedTheme }}

<m3-radio-button
  name="theme"
  value="light"
  [checked]="selectedTheme === 'light'"
  (radio-change)="onRadioChange($event, 'theme')">
</m3-radio-button>

Interactive Examples

Radio buttons emit events when their state changes, allowing you to react to user selections.

States

Radio buttons can be in different states: enabled, disabled, checked, and unchecked.

Disabled State

Disabled radio buttons cannot be interacted with and have reduced opacity.

<m3-radio-button disabled></m3-radio-button>
<m3-radio-button checked disabled></m3-radio-button>

Accessibility

The radio button component follows Material Design 3 accessibility guidelines:

ARIA Support

  • ✓ Uses proper ARIA attributes (role="radio", aria-checked, aria-disabled)
  • ✓ Supports keyboard navigation (Space and Enter keys)
  • ✓ Provides focus indicators
  • ✓ Supports screen readers with aria-label and aria-labelledby
  • ✓ Properly manages radio groups
<m3-radio-button aria-label="Option A"></m3-radio-button>
<m3-radio-button aria-labelledby="label-id"></m3-radio-button>

Properties & Events

Properties

Property Type Default Description
checked boolean false Whether the radio button is checked
disabled boolean false Disables the radio button
name string null Name attribute for radio group (required for grouping)
value string null Value attribute for form submission
aria-label string null ARIA label for accessibility

Events

Event Detail Description
radio-change { checked: boolean, name: string | null, value: string | null } Fired when the radio button state changes
