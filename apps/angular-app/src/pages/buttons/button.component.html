Material Design 3 Buttons

Buttons help people take action, such as sending an email, sharing a document, or liking a comment.

Button Variants

Material Design 3 provides five button types to support different emphasis levels:

Filled Button

High Emphasis

The most prominent button type. Use for the primary, most important action on a screen.

Filled Button Disabled
<m3-button variant="filled">Filled Button</m3-button>

Elevated Button

Medium-High Emphasis

Adds dimension to mostly flat layouts. Use for secondary actions that need separation.

Elevated Button Disabled
<m3-button variant="elevated">Elevated Button</m3-button>

Tonal Button

Medium Emphasis

An alternative middle ground between filled and outlined buttons. Good for secondary actions.

Tonal Button Disabled
<m3-button variant="tonal">Tonal Button</m3-button>

Outlined Button

Medium Emphasis

For actions that need attention but aren't the primary focus. Often paired with filled buttons.

Outlined Button Disabled
<m3-button variant="outlined">Outlined Button</m3-button>

Text Button

Low Emphasis

For the least important actions. Often used in cards, dialogs, and toolbars.

Text Button Disabled
<m3-button variant="text">Text Button</m3-button>

â™¿ Accessibility Features

  • Keyboard Navigation: Full support for Enter/Space activation and Tab focus
  • Focus Indicators: Visible focus rings for keyboard users
  • ARIA Support: Proper aria-label, aria-busy for loading states
  • Touch Targets: Minimum 48x48px tap targets (includes invisible padding)
  • Screen Readers: Disabled and loading states properly announced
  • Color Contrast: WCAG 2.1 Level AA compliant color combinations
  • Flexible Sizing: Five size options (extra-small to extra-large) for optimal touch targets
  • Shape Morphing: Dynamic visual feedback when pressed improves interaction clarity
