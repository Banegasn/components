Quick Start

Learn how to easily integrate Material 3 Web Components into any project.

⚡ No Build Step (via CDN)

The fastest way to use the components is by importing them directly in your HTML using the jsDelivr CDN and ES modules. This requires no build step and works natively in modern browsers.

📱 Live Demo

📦 Install via Package Manager

For more complex applications or when using a framework, you can install the components individually from npm.

npm

pnpm

🅰️ Angular

In Angular, make sure to add CUSTOM_ELEMENTS_SCHEMA to your component or module to prevent template errors for unknown tags.

⚛️ React

React 19+ natively supports Custom Elements. For listening to custom events, prefix them with on (e.g., onbutton-click).

💚 Vue

Vue has native support for Custom Elements. You can use standard Vue directives like @button-click to listen to custom events.
