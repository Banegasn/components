Split Button

A split button groups a primary action with a related menu of options. Material 3 Expressive adds shape morphing and rotation animations.

Variants

Split buttons come in the same variants as standard buttons: filled, tonal, elevated, and outlined.

Usage

Use split buttons when you have a primary action and related secondary actions. The left side triggers the primary action, while the right side (arrow) opens a menu.

Common Use Cases

  • Send / Schedule Send: Primary action sends immediately, menu offers scheduling.
  • Save / Save As: Primary action saves, menu offers formats.
  • Merge / Squash & Merge: GitHub-style pull request actions.

♿ Accessibility

  • Keyboard Navigation: Tab to focus, Enter/Space to activate.
  • Screen Readers: The two parts are semantically separate buttons.
  • ARIA: The menu trigger has aria-haspopup="menu" and aria-expanded states.
